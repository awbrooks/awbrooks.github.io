Fuzzy String Matching¶

Python functions to compare strings built on the *fuzzywuzzy* and *Levenshtein* package algorithms.

Description¶

Function Groups:
  • str_fuzzy_match_* - Functions to find best matching strings from a list to a query string.
  • str_fuzzy_score_* - Functions to calculate a score on how similar strings are.
Function Specialization:
  • *_full - Takes full words, punctuation, and spaces into account.
    • Test how similar two complete strings are.
  • *_partial - Takes full words, punctuation, and spaces into account; removes prefix and suffix
    • Tests if one string appears in other string.
